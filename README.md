# database_wow



## Description
This project is a simple C-based application developed as a university assignment.  
It simulates working with a basic database structure inspired by World of Warcraft.

The goal of the project is to practice fundamental programming concepts, data handling, and structuring program logic in C.

---

## Features
- Basic data storage and management
- Simple operations on structured data
- Console-based interaction
- Practice of C programming fundamentals

---

## Technologies
- C programming language

---

## Purpose
Educational project created to improve understanding of programming basics, data structures, and application design in C.
